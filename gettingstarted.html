Getting Started

Prerequisites

Java (for closure compiler)

Ruby (for compass)

Compass

Install

git clone https://github.com/ThoughtLeadr/Sasquatch.git

Go into the sasquatch directory and run "sudo python setup.py install"

Create an app

"sasquatch create-project <name>"

Go into the project folder that was just created and run "sasquatch dev server"

Point your borwser at localhost:8000/bin

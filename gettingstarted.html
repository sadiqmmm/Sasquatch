Getting Started

Prerequisites

  1. Java: Required for the Google Closure Compiler, used to minify CSS and Javascript in production builds.
  2. Ruby: Required for Compass Styling Framework.
  3. Compass: Required to support SCSS and advanced styling functionality.

Install

Run the following commands from your terminal in a location where you'd like the Sasquatch Open Source project to reside on your system.

  1. git clone git@github.com:ThoughtLeadr/Sasquatch.git
  2. cd Sasquatch
  3. sudo python setup.py install

Sasquatch is setup and ready to go!

Create an app

To create your first Sasquatch application, open up a terminal in a directory you'd like the project in.

  1. sasquatch create-project myproject
  2. cd myproject
  3. sasquatch dev server

Open up your browser to localhost:8000/bin and you'll have see your very first Sasquatch Application. To learn how to use all the functionality available in Sasquatch, you should check out our Todo application.

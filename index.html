What is Sasquatch?

Sasquatch makes it incredibly simple to build your own desktop calibre webapps. Forget about spending time learning a large, complex view object model. You already know jQuery, right? Of course you do. Use jQuery's rich plugin ecosystem & readable HTML templates to create baller webapps using Sasquatch.

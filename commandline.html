Commandline Options

$ sasquatch create-project <name>

creates new project with a couple example views and controllers

$ sasquatch create-view <name>

creates empty view and controller with name

$ sasquatch dev

builds files into bin directory and watches for changes

$ sasquatch dev server

builds files into bin directory and watches for changes, also starts a simple server to serve files from localhost:8000/bin

$ sasquatch dev once

runs the dev build once and quits

$ sasquatch prod

creates production build minifing css and js (using closure compiler)
